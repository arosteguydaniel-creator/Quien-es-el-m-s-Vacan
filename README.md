"Quien es el más Vacán? "(Who is the G.O.A.T) 🐮✨
This is a fun, fast-paced "True or False" trivia game designed to challenge your general knowledge while keeping you entertained. It's built as a lightweight, interactive web application.

🚀 Overview
The project is an interactive web game that presents a series of random facts. Users must test their knowledge by guessing whether a statement is "True" or "False" before revealing the answer. It features smooth CSS animations, a responsive design, and a celebratory confetti finish.

🛠️ Technologies Used
HTML5: Core structure.

CSS3: Styling, @keyframes animations, and responsive layout.

JavaScript: Game logic, question bank management, and state control.

Canvas Confetti: External library for the victory celebration effect.

📦 Key Features
Dynamic Question Bank: Includes a diverse array of fun, verified facts.

Engaging Interface: Animated gradient background and a "floating" mascot animation.

User Experience (UX):

On-demand answer reveal.

Real-time question counter.

Victory celebration (confetti.js) upon completion.

Responsive: Fully optimized for mobile and desktop screens.

⚙️ How to Run
Clone this repository or download the index.html file.

Ensure you have an image file named VACAN_2.png in the same directory (the code will gracefully hide it if the image is missing).

Open index.html in any web browser. No server setup is required!
